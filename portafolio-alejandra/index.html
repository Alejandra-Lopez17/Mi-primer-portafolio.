Alejandra Lopez Soy una desarrolladora Front-End Developer con experiencia en el desarrollo de aplicaciones web y sitios interactivos. Me especializo en el uso de tecnologías como HTML, CSS, JavaScript para crear experiencias de usuario atractivas y funcionales.
